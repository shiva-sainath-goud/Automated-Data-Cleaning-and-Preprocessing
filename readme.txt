# Automated Data Cleaning & Preprocessing Tool

## 📌 Project Description
This project is a Streamlit-based web application for automated data cleaning and preprocessing. It provides functionalities such as handling missing values, removing duplicates, outlier detection, normalization, scaling, and categorical encoding. The app also offers data visualization before and after cleaning.

## 🚀 Features
- Upload and preview raw data
- Handle missing values
- Remove duplicates
- Detect and remove outliers
- Normalize text formatting
- Scale numeric features
- Encode categorical variables
- Visualize data before and after cleaning
- Download cleaned dataset

## 🛠 Tech Stack
- **Frontend:** Streamlit
- **Backend:** Pandas, NumPy, SciPy, Scikit-Learn
- **Visualization:** Matplotlib, Seaborn

## ⚡ Installation & Usage

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/Automated-Data-Cleaning-Tool.git
cd Automated-Data-Cleaning-Tool
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application
```bash
streamlit run app.py
```

### 4️⃣ Open in Browser
Once the server starts, it will provide a local URL (e.g., `http://localhost:8501`). Open it in your browser to use the tool.
